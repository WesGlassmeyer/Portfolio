BJJ Sparring Tracker
Github Repo

In Brazilian Jiu-Jitsu we do a lot of sparring. You can use this app to track your sparring sessions and use the statistics to help guide your training. As a BJJ instructor, I wanted to create this app to help BJJ students progress faster. The app utilizes React, CSS, Jest, Vercel, and Javascript. The backend for this project was built using Node, Express, and Postgresql.

BJJ Symposium
Github Repo

This app allows you to search for Brazilian Jiu-Jitsu videos from YouTube. If you find a video you think is valuable, you can rate it, apply tags and commit it to the Community Favorites Page. I wanted to create this app as a sort of filter for the tremendous amount of available BJJ instructional videos. Practitioners new to the sport may not be able to tell the good from the bad, and I hope this resource can help. The app utilizes React, Jest, Vercel, HTML, CSS, Javascript, and makes calls to YouTube's API. We also built the backend for this project using Node, Chai, Mocha, Postgresql, and Knex.
