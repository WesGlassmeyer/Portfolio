BJJ Sparring Tracker
Github Repo

In Brazilian Jiu Jitsu we do a lot of sparring. You can use our app to track your sparring sessions and use the statistics to help guide your training. The app utilizes React, CSS, Jest, Vercel, and Javascript. The backend for this project was built using Node, Express and Postgresql.

BJJ Symposium
Github Repo

Our app allows you to search for Brazilian Jiu Jitsu videos from YouTube. If you find a video you think is valuable, you can rate it, apply tags and commit it to the Community Favorites Page. The app utilizes React, Jest, Vercel, HTML, CSS, Javascript, and makes calls to YouTube's API. We also built the backend for this project using Node, Chai, Mocha, Postgres and Knex.

Workout Generator
